\doxysubsection{Detailed Description}
Manages multiple drones through \mbox{\hyperlink{classDroneController}{Drone\+Controller}} instances. 

This class provides a high-\/level interface for managing multiple drones. It handles the creation and lifecycle of \mbox{\hyperlink{classDroneController}{Drone\+Controller}} instances, each of which manages a single drone. The class provides methods for controlling the drones and managing their states.
