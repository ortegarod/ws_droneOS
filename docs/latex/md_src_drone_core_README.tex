{\bfseries{drone\+OS}} is a proprietary, ROS 2-\/based modular SDK for commanding PX4-\/compatible autonomous drone fleets. It provides reusable mission logic, low-\/level PX4 communication, and real-\/time control infrastructure for scalable, multi-\/agent operations.\hypertarget{md_src_drone_core_README_autotoc_md1}{}\doxysection{‚úàÔ∏è Features}\label{md_src_drone_core_README_autotoc_md1}

\item Fleet-\/level drone management via {\ttfamily \mbox{\hyperlink{classDroneAgent}{Drone\+Agent}}} and {\ttfamily \mbox{\hyperlink{classDroneManager}{Drone\+Manager}}}
\item YAML-\/driven waypoint missions with auto-\/progress logic
\item Real-\/time position tracking using PX4 telemetry
\item Modular architecture designed for expansion into AI-\/driven command layers
\item Compatible with PX4 Autopilot and ROS 2 middleware
\item Thread-\/safe drone state management
\item Configurable mission parameters and behaviors
