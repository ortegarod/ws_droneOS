droneOS

droneOS is a proprietary, ROS 2-based modular SDK for commanding PX4-compatible autonomous drone fleets. It provides reusable mission logic, low-level PX4 communication, and real-time control infrastructure for scalable, multi-agent operations.

The SDK is built around a core set of components:

  • DroneAgent: Individual drone control and state management
  • DroneManager: Fleet-level coordination and mission execution
  • Mission Parser: YAML-based mission configuration
  • PX4 Interface: Low-level communication with PX4 autopilot

üöÄ Getting Started

Requirements

  • ROS 2 Humble or newer
  • PX4 Autopilot + px4_msgs
  • yaml-cpp and standard ROS 2 build tools
  • C++17 compatible compiler

Dependencies

# Install ROS 2 dependencies
sudo apt install ros-${ROS_DISTRO}-px4-msgs
sudo apt install libyaml-cpp-dev

Build

cd ~/ws_droneOS
colcon build --packages-select drone_os
source install/setup.bash

Usage

  1. Launch the drone core:
    ros2 launch drone_os drone_core.launch.py
  2. Configure missions using YAML files (see examples in config/)
  3. Use the ROS 2 API to:
    • Monitor drone states
    • Send commands
    • Manage missions
    • Coordinate fleet operations
